# RecordAR - App para Adultos Mayores

## 📱 Descripción
App mobile diseñada para ayudar a adultos mayores con tareas cotidianas:
- 📝 Notas simples
- 📅 Eventos con recordatorios
- 🎮 Tutoriales para ejercitar la memoria

## 🚀 Comenzar

### Prerrequisitos
- Node.js v16+
- Expo CLI
- Cuenta en GitHub

### Instalación
1. Clonar el repositorio
2. `npm install`
3. `npx expo start`
